# Roadmap

## v0.1.0 - CLI Release

命令行安装版，目标是稳定完成无人值守自动认证。

- 交互式配置校园网登录 URL、账号和密码
- 使用 Windows DPAPI 机器级加密保存密码
- 安装 Windows 开机计划任务
- 以 `SYSTEM` 身份开机自动运行
- 自动检测联网状态
- 自动提交上海大学 ePortal 认证
- 提供日志和手动测试命令

## v0.1.1 - CLI Patch

命令行安装版补丁更新。

- 密码输入为空时阻止继续配置
- 支持自动访问 `10.10.9.9` 探测 ePortal 登录 URL
- 补充适用网络环境说明
- 优化下载和安装路径说明

## v0.1.2 - CLI Patch

命令行安装版补丁更新。

- 配置阶段不再要求当前 ePortal 登录页长 URL
- 开机任务在真正需要认证时实时访问校园网关获取登录参数
- 保留备用完整 ePortal 登录 URL 配置能力
- 优化已在线用户的配置体验

## v0.2.0 - Desktop UI Release

图形界面版，目标是降低非命令行用户的部署门槛。

- 图形化账号配置
- 一键安装/卸载开机任务
- 显示任务状态
- 显示最近日志
- 手动测试认证
- 复用 v0.1.0 的核心认证逻辑
